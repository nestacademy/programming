## Solution 1.
###Straight-forwart implementation using nested-loop.

```
void solve(int n) {
    for(int  i = 1; i <= n; i++) {
        for(int j=0; j < i; j++) {
            cout << i;
        }
        cout << endl;
    }
}
```
## Solution 2.
### Using small math trick(10^n-1=99..999) implementation.

```
void solve(int n) {
    for (int i = 1; i <= n; i++) {
        pow10 = pow(10, i);
        ones = (pow10 - 1) / 9;
        cout << ones * i << endl;
    }
}
```
